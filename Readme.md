## Bee

### Feature

* Global
* Dynamic Watch
* Zero Configuration
* `Middleware`

### Usage

#### Init 

* Enter your project path, Such as `/usr/username/path/to/repo`
* bee init <name>
    It will generate structure like that: 

    ```
    ├── example
    │   └── index.js
    ├── package.json
    └── src
        └── index.js
    ```
* cd <name>

#### Start

> The start cli will run the file inside the example folder.

**All** `yarn start`
**Optional** `yarn start <name>`
